# Déploiement de Yoga In Lyon

Voici les fichiers sources du site web statique optimisé pour le SEO de votre annuaire lyonnais.

## Structure du projet :
- `index.html` : Page d'accueil avec textes SEO enrichis (> 250 mots).
- `style.css` : Feuille de style.
- `script.js` : Carte interactive Leaflet.
- `clubs/` : Contient 12 pages uniques générées pour chaque studio, avec contenu enrichi.

## Mise en ligne :
Ce site est composé de fichiers purement statiques. Vous pouvez l'héberger sur n'importe quel service d'hébergement classique ou Cloud (ex: OVH, Hostinger, Vercel, Netlify, Cloudflare Pages, etc.). Il suffit d'envoyer ces fichiers à la racine de votre serveur web public.
